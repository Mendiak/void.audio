# void.audio 📼
### // Terminal-based Retro-Futuristic Signal Processor

**void.audio** is a high-fidelity, browser-based music player designed with a brutalist, retro-futuristic aesthetic. It mimics a specialized terminal terminal for signal analysis and audio reproduction, featuring hardware-inspired UI elements, ASCII visualizations, and multi-spectrum themes.

## 🛠 System Capabilities

### 1. Signal Archive (Library)
- **Bulk Injection**: Inject multiple local audio files (MP3, WAV, FLAC) directly into the archive.
- **Hardware Indicators**: Active tracks feature pulsing LED status lights and real-time activity icons.
- **Technical Metadata**: Automated extraction of file size, type, and source origin.

### 2. Neural Visualizer
- **ASCII Rendering**: Real-time FFT (Fast Fourier Transform) analysis rendered through a custom ASCII engine.
- **Discrete Signal Path**: Stereo visual representation of audio peaks and frequencies.

### 3. Multi-Spectrum Interface
- **Dynamic Themes**: Switch between multiple specialized color spectrums (Toxic Green, Cyber Blue, Volcanic Red, Amber Alert).
- **CRT Aesthetics**: High-fidelity scanlines, glitch effects, and monospace typography for maximum immersion.

### 4. Diagnostic Suite
- **System Stats**: Monitor FFT resolution, interface latency, and buffer status.
- **Signal Identity**: Dedicated slot for signal origin and metadata verification.

- **Icons**: [Lucide React](https://lucide.dev/)

## 🔌 Local Installation

To initialize the node in your local environment:

```bash
# 1. Clone the repository
git clone https://github.com/Mendiak/void.audio.git

# 2. Enter the directory
cd void.audio

# 3. Install dependencies
npm install

# 4. Start the engine
npm run dev
```
